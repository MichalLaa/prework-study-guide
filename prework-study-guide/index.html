HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • How to build frame of our webpage.

CSS

  • Stylesheet language used to describe the presentation of a document written in HTML.
  • Create the actual content of the page.
  • By implementing CSS to HTML code we give a design or style to our webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Version control system used for tracking changes in computer files.
  • GitHub Lets you and others work together on projects from anywhere.
  • Git and GitHub it's a platform where we can work with a group of people in our offices or anywhere where they are based.

javascript

  • Programming language used by developers to make web pages interactive.
  • Allows developers to create dynamically updating content, use animations, pop-up menus, clickable buttons, control multimedia, etc
  • In different words Javascript gives life to our website by all animations, dynamucally content,buttons, etc.
